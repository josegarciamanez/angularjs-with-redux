# Sandbox

# Getting started
- Clone this repo
- nvm use
- yarn install
- yarn run devserver // this start de webpack-dev-server
- Go to http://localhost:8080/

# Included libraries
- AngularJs 1.6.4
- lodash 4.17
- webpack 4.10
- webpack-dev-server 3.1.4
- style-loader, css-loader
- ng-redux 4.1.0-alpha
